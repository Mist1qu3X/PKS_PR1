# Практическое занятие №1: Программирование корпоративных систем
## Тема: Управление библиотекой книг (WPF + Entity Framework Core + PostgreSQL)

## Описание проекта

Разработано приложение для управления библиотекой книг с использованием:
- **WPF** (Windows Presentation Foundation) — для пользовательского интерфейса
- **Entity Framework Core** — для работы с базой данных
- **PostgreSQL** — в качестве СУБД

Приложение реализует полноценное управление библиотекой с возможностью хранения информации о книгах, авторах и жанрах.


## Выполненные требования

### 1. Модели данных (сущности)
**Book** (Id, Title, PublishYear, ISBN, QuantityInStock)
**Author** (Id, FirstName, LastName, BirthDate, Country)
**Genre** (Id, Name, Description)

### 2. Связи между сущностями
- Книга может иметь одного автора (связь один-ко-многим)
- Книга может относиться к одному жанру (связь один-ко-многим)

### 3. DbContext и Fluent API
- Настроены первичные ключи
- Настроены обязательные поля
- Установлены ограничения на длину строковых полей
- Настроено каскадное удаление

### 4. WPF интерфейс
- **Главное окно** с DataGrid для отображения книг
- **Панель фильтрации** по жанру и автору
- **Кнопки** для добавления, редактирования и удаления книг
- **Окно добавления/редактирования** книги
- **Окно управления авторами**
- **Окно управления жанрами**

### 5. Функционал
- CRUD операции для всех сущностей
- Поиск книг по названию
- Фильтрация книг по автору и жанру
- Отображение общего количества книг в наличии
